# webserver-clj-template
template for a clojure prject with webserver

## Dependencise

### just

https://github.com/casey/just

## Features

The following features are added to the template

* kaocha
* aot
* uberjar

## Run tests

```
bin/kaocha
```

## aot

```
just aot
```

## Uberjar

### Creating

```
just uberjar
```

### Running

```
java -jar target/*.jar
```
